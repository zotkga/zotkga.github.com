이 사이트의 현재 버전은 수행평가 제출용이며, 로컬 호스트에서 열립니다.

방명록, 댓글과 같은 인터넷이 필요한 기능은 도메인을 등록한 후 이용 가능하게 할 예정입니다.

2024. 6. 17.


게임 설명

이 게임은 자료구조 수행평가를 위해 제작되었습니다.

중위 표기식을 후위 표기식으로 변환하는 간단한 게임입니다.


코드 설명

주요 기능 설명
스택 생성 및 초기화:
스택을 초기화하고 공백 상태인지, 포화 상태인지 확인합니다.

스택 조작 함수:
스택에 요소를 삽입하고 삭제하는 함수입니다.

우선순위 확인 함수:
연산자의 우선순위를 확인합니다.

중위표기식을 후위표기식으로 변환하는 함수:
중위표기식을 후위표기식으로 변환합니다.

랜덤 중위표기식 생성 함수:
무작위 중위표기식을 생성합니다.

게임 시작 함수:
게임의 메인 루프를 관리하며, 사용자의 입력을 받아 정답을 확인하고 하트를 관리합니다.
메인 함수에서는 start_game 함수를 호출하여 게임을 시작합니다.
