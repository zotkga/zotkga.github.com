인터넷에 연결되지 않았습니다.

이 사이트의 현재 버전은 수행평가 제출용이며, 로컬 호스트에서 열립니다.

방명록, 댓글과 같은 인터넷이 필요한 기능은 도메인을 등록한 후 이용 가능하게 할 예정입니다.

2024. 6. 17.



게임 설명

이 게임은 파이썬 수행평가를 위해 제작되었습니다.

부평역 지하상가를 탈출한다는 컨셉의, 텍스트 기반 던전 게임입니다.



코드 설명

코드 보기

주요 클래스 및 함수
한글 조사 출력 함수
이 함수는 한국어 문법에 맞게 조사를 출력하기 위해 사용됩니다.
종성(kstr): 문자열의 마지막 글자가 받침을 가지는지 확인합니다.
을(kstr), 이(kstr), 와(kstr), 은(kstr): 각각 을/를, 이/가, 과/와, 은/는 조사를 올바르게 붙여줍니다.

플레이어 클래스
플레이어의 기본 정보(이름, 최대 HP, 현재 HP, 골드, 인벤토리, 최소/최대 공격력)를 저장합니다.
공격(): 몬스터를 공격합니다.
방어(): 방어 태세를 취합니다.
아이템_사용(): 인벤토리에서 아이템을 사용하여 HP를 회복합니다.

몹 클래스
몬스터의 기본 정보(이름, 최대 HP, 현재 HP, 골드, 스킬)를 저장합니다.
스킬_사용(): 몬스터가 스킬을 사용하여 플레이어를 공격합니다.



주요 데이터
몬스터 데이터
깡패, 잼민이, 일진, 할머니, 바가지상인, 노숙자, 자전거배달원: 각각 다른 이름과 스킬을 가진 몬스터 객체입니다.
몹리스트: 각 길에 등장하는 몬스터 리스트를 저장합니다.

아이템 데이터
껌, 초코바, 아이스크림, 몬스터에너지, 즉석떡볶이, 삼각김밥, 술, 매운닭발, 하겐다즈: 각각 이름, 설명, 가격, 회복량을 가진 아이템 객체입니다.
아이템리스트: 상점에서 등장할 수 있는 아이템 리스트를 저장합니다.


주요 함수
체력바()
현재 체력과 최대 체력 비율에 따라 체력바를 출력합니다.
전투()
플레이어와 몬스터 간의 전투를 진행합니다.
상점()
상점을 발견했을 때, 아이템을 구입할 수 있는 기회를 제공합니다.
기타_이벤트()
다양한 랜덤 이벤트를 발생시킵니다.


게임 진행
게임은 플레이어가 부평역 지하상가를 탐험하며 몬스터와 싸우고, 아이템을 사용하여 HP를 회복하고, 최종적으로 목표 지점에 도달하면 승리하는 방식으로 진행됩니다.



게임 플레이

코드를 다운로드하고, idle이나 powershell로 실행하시면 됩니다.

py "지하상가 탈출하기 0.1.1.py"

실행에는 python 설치가 필요합니다.
